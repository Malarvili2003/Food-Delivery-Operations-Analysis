#  Food Delivery Operations Analysis Dashboard  

##  Project Overview  

This project analyzes food delivery operations using **Power BI** to identify performance gaps, delivery delays, and rating trends.  

The objective of this dashboard is to transform raw operational data into actionable business insights that improve efficiency and customer satisfaction.

---

##  Key Dashboard Metrics  

The dashboard provides insights into:

- **Total Orders**
- **Average Delivery Time**
- **Average Delivery Rating**
- **On-Time vs Late Delivery Percentage**
- **Order Distribution by City**
- **Traffic Impact on Delivery Time**
- **Vehicle-wise Delivery Efficiency**
- **Weather Impact on Delivery Performance**

---

##  Tools & Technologies Used  

- **Power BI** – Data Visualization  
- **DAX (Data Analysis Expressions)** – KPI & Calculated Measures  
- **Power Query** – Data Cleaning & Transformation  
- **CSV Dataset** – Operational Data Source  

---

##  Data Cleaning & Transformation  

The following preprocessing steps were performed:

- Removed null values and duplicates  
- Standardized categorical variables  
- Converted delivery time into numeric format  
- Formatted date columns  
- Created calculated measures using DAX  

---

##  Key KPIs Created  

- Total Orders  
- Average Delivery Time  
- Average Delivery Rating  
- Late Orders Percentage  
- Orders by City  
- Orders by Traffic Condition  
- Delivery Time by Vehicle Type  

---

##  Business Insights  

- Delivery time increases significantly during **high traffic conditions**.  
- Lower customer ratings are associated with **longer delivery times**.  
- Peak ordering hours were identified for **workforce optimization**.  
- Certain cities demonstrate **higher operational efficiency**.  
- Multiple deliveries significantly increase delivery time.  

---

##  Business Recommendations  

- Optimize delivery partner allocation based on distance and traffic conditions.  
- Increase workforce capacity during identified peak hours.  
- Introduce incentive programs for high-performing delivery agents.  
- Improve route planning using traffic trend analysis.  
- Monitor late order percentage to enhance customer satisfaction.  

---

##  Project Outcome  

This dashboard enables data-driven decision-making by providing operational visibility across delivery performance, revenue distribution, and workforce efficiency.  

The project demonstrates practical skills in **data analysis, KPI development, DAX modeling, and business intelligence reporting.**
