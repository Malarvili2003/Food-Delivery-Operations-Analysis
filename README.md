# Food Delivery Operations Analysis Dashboard

## Project Overview

This project analyzes food delivery operations using Power BI to identify performance gaps, delivery delays, and rating trends.

The dashboard provides insights into:

* Total Orders
* Average Delivery Time
* Delivery Partner Ratings
* Order Distribution
* On-Time vs Late Deliveries

---

## Tools & Technologies Used

* Power BI
* DAX (Data Analysis Expressions)
* CSV Dataset
* Data Cleaning & Transformation

---

## 📊 Key KPIs Created

* Total Orders
* Average Delivery Time
* Average Delivery Rating
* On-Time Delivery Percentage
* Orders by City
* Orders by Traffic Condition

---

## Business Insights

* Delivery time increases significantly during high traffic conditions.
* Lower ratings are associated with longer delivery times.
* Peak ordering hours identified for workforce optimization.
* Certain cities show higher operational efficiency.

---

## Business Recommendations

* Optimize delivery partner allocation based on distance.
* Increase delivery partners during peak hours.
* Implement incentive program for high-rated delivery agents.
* Improve route planning to reduce delays.
